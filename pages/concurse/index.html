Concurso Trupper

Truper por el año 2019 organizó un concurso para desarrolar algo tecnológico autosustentable para ayudar a las comunidades. Este concurso se llevó acabo en el municipio de Jilotepec, Estado de México.

La mi propuesta tecnológica y autosustentable para este concurso fue un sistema automático para cuidar y monitorear automaricamente cultivos. Este se podría usar tanto a grán escala para invernaderos grandes y empresas que se dedican a esto, como a pequeña escala teniendo tu propia huerta en casa.

El sistema que desarrolle consiste en una serie de sensores, sistemas de riego y calefacción conectados una central o microcontrolador en el cuál se toman ciertas mediciones de los sensores como temperatura, humedad y calidad del aire y con base en esos datos el sistema tomaría desiciones para cuidar los cultivos.

Esta información se veía en una app o página web con un estilo de panel de control donde veías a todas las plantas que tienes cuidando y su edad.

Solo se elaboró un prototipo, con unos focos se simulaba el sistema de calefacción y una bomba pequeña de agua simulaba el sistema de riego. Dependiendo la planta, ésta debía estar a cierta temperatura, supongamos una papaya. Las payas son de calor, entonces si la temperatura bajaba demasiado se encedían los focos (sistema de calefacción), si la temperatura ya estaba correcta los focos se apagaban. Si la humedad era poca, la papaya necesitaba agua, entonces se encendía el motor y comenzaba a caer agua.

Básicamente ese era el funcionamiento del sistema. Ganamos el segundo lugar del consurso pues la exposición no salio del todo bien. Tenía más ideas para agregarle, pero por tiempo es todo lo que se pudo hacer.

Para este proyecto utilice C++, Arduino, HTML, CSS, JavaScript y SQL.
